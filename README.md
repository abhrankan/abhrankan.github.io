# abhrankan.github.io

> Static systems. Hidden networks. Signal over noise.

Personal index for Tor hidden services, experimental web projects, and interactive signal environments.

---

## Manifesto

Most websites become more complicated over time.

These projects move in the opposite direction.

- No tracking
- No analytics
- No accounts
- No unnecessary JavaScript
- Static where possible
- Minimal always

The goal is not scale.

The goal is signal.
