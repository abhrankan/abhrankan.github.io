# Abhrankan Chakrabarti – Portfolio Website

This repository hosts my personal portfolio website, presenting selected work in mathematics, algorithm design, and computational problem-solving.

## Overview

The website includes:

- An embedded **Sudoku Solver** (via GitHub Gist)  
- Algorithmic and number-theoretic research projects  
- Interactive computational tools  
- A minimal, responsive design focused on clarity
