# Abhrankan Chakrabarti

This repository contains the source code for my personal portfolio website.

The site presents selected work in:

- Number theory
- Algorithm design
- Numerical computation
- High-precision mathematical methods

## Featured Projects

- **[Winternship 2025 Case Studies](https://github.com/Abhrankan-Chakrabarti/Internship_caseStudies)**  
  Full MERN stack training — TypeScript, React (Vite), MongoDB (ACID transactions), and Express.js REST APIs with MVC architecture.

- **[LlamaInteract](https://github.com/Abhrankan-Chakrabarti/LlamaInteract)**  
  Local LLM interaction system built using llama.cpp for efficient on-device inference.

- **[Catalan π Algorithm](https://github.com/Abhrankan-Chakrabarti/pi-calculation-with-catalan-numbers)**  
  A computational method for approximating π using Catalan numbers.

- **[Almost Isosceles Pythagorean Triples](https://github.com/Abhrankan-Chakrabarti/almost-isosceles-pythagorean-triples)**  
  Algorithmic exploration of structured Diophantine patterns.

- **[VisionaryAI (GeminiFusion)](https://github.com/Abhrankan-Chakrabarti/GeminiFusion)**  
  AI-integrated web application combining generative models.
